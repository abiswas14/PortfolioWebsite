# Aalif Biswas Portfolio

Dark, animated personal portfolio website built with React, Vite, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This works well on Vercel, Netlify, or GitHub Pages. For Vercel/Netlify, import the repository and use:

- Build command: `npm run build`
- Output directory: `dist`
